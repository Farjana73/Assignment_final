# Project Name - Blog Project with Dashboard

A simple blog creating website with **React**, **Node.js**, **Express**, and **MongoDB** to create, view, and manage blogs effectively.

---

## ЁЯУЦ Features
- ЁЯУЪ View all blogs with basic information.
- ЁЯУЭ Detailed blog page for each blog.
- ЁЯЦЛя╕П Add, edit, and delete blogs.
- тЬЕ Contact Message and Mark Message as `Seen` or `Unseen`.
- ЁЯУК Message status counter (e.g., seen/unseen).

---

## ЁЯЫая╕П Tech Stack
- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)

---

## ЁЯЪА Installation

Follow these steps to run the project locally:



1. Install Dependencies

Backend:
cd backend
npm install

Frontend:
cd frontend
npm install

2. Run the Application

Backend:
cd backend
npm run dev

Frontend:
cd frontend
npm run dev
